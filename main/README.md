## ファイル機能

- main.ino : ```loop関数で関数呼び出し```

- name_main.ino :
```loop関数で呼び出す関数を定義。名前ごとに動作が異なる(予定)```

- lightlib.ino : ```ライトアップ用のライブラリ関数を保存したファイル。ここからname_main.inoで使う関数を選んでもろて```
